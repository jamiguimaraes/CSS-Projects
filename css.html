Aprendendo "o mínimo" de CSS

CSS é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML (incluindo várias linguagens de marcação como SVG, MathML ou XHTML). O CSS descreve como os elementos devem ser renderizados na tela, no papel, na fala ou em outras mídias.

CSS é uma das tecnologias centrais da World Wide Web, juntamente com HTML e JavaScript.
